What's OwlEyes?

UI display issues such as text overlap, missing image always occur during GUI rendering on difference devices due to the software or hardware compatibility. They negatively influence the app usability, resulting in poor user experience. To detect these issues, we provide the OwlEyes, based on deep learning for modelling visual information of the GUI screenshot. OwlEyes can detect GUIs with display issues and also locate the detailed region of the issue in the given GUI for guiding developers to fix the bug.

OwlEyes is named as our tool is like the owl's eyes to effectively spot UI display issues. And our tool (nocturnal like owl) can complement with conventional automated GUI testing (diurnal like eagle) for ensuring the robustness of the UI.

OwlEyes is capable of...

Running an app online

Users only need to provide APK download link or APK file.

Collecting screenshots

OwlEyes can automatically run the application and get the run-time GUI screenshot.

UI display issues detection

OwlEyes can automatically detect whether there is an issue in the screenshot.

UI display issues localization

OwlEyes can localize the regions with UI display issues in the screenshots for guiding developers to fix the bug.

Details about our method

We proposes OwlEyes to automatically detect and localize UI display issues in the screenshots of the application under test. Given one APK or UI screenshot, our tools will automatically run the application and get its UI screenshots. Then our CNN-based model can first classify if it relates with any display issues via the visual understanding. Once the issue is confirmed, our model can further localize the detailed issue position on the UI screenshot by Grad CAM-based model for guiding developers to fix the bug.

Automatically get UI screenshots

We use dynamic analysis tools to automatically explore applications and get UI screenshots.

CNN-based Issues Detection

We adopt the convolution neural network, including 12 convolutional layers, 6 pooling layers and 4 fully connection layers.

Grad CAM-based Issues Localization

We adopt the visualization method of CNN.

  • produce a localization map of the important regions

  • obtain the Grad-CAM positioning

  • the point multiplication with the back propagation
