What's OwlEyes?

UI display issues always occur during GUI rendering on difference devices.
UI display issues negatively influence the app usability, resulting in poor user experience.
OwlEyes is based on deep learning for modelling visual information of the GUI screenshot.
OwlEyes can detect, locate display issues of GUI and guide developers to fix those bugs.

Some Screenshots With UI Display Issue

Textual information or component is occluded by other components.

In the icon position, the image is not showing as its design.

The right information is not displaying, instead NULL is showing in corresponding area.

Pieces of text are overlapped with each other.

The screen is blurred.

Details about our method

#} {# #} {#

#} {# We proposes OwlEyes to automatically detect and localize UI display issues in the screenshots of#} {# the#} {# application under test. Given one APK or UI screenshot, our tools will automatically run the#} {# application#} {# and get its UI screenshots. Then our CNN-based model can first classify if it relates with any#} {# display#} {# issues via the visual understanding. Once the issue is confirmed, our model can further localize#} {# the#} {# detailed issue position on the UI screenshot by Grad CAM-based model for guiding developers to#} {# fix the#} {# bug.#} {#
